# Hailey Bui Portfolio Website

This is a static one-page portfolio website built in a White Editorial style.

## Files
- `index.html`
- `style.css`
- `script.js`
- `assets/`
- `VIDEO_LINKS_TO_ADD.md`

## How to preview locally
Open `index.html` in your browser.

## How to host for free
Recommended:
1. Create a GitHub repository.
2. Upload this whole folder.
3. Deploy with Vercel Free or Netlify Free.

## Where to edit video links
Open `VIDEO_LINKS_TO_ADD.md`, fill in your video URLs, then replace the placeholder `href="#"` links in `index.html`.

## Design direction
White background, black editorial typography, subtle red accents, recruiter-friendly layout for hospitality/lifestyle marketing roles.
